# Cook Project

这是一个前后端分离的项目。

## 目录结构

- `frontend/`: React (Vite + TypeScript) 前端项目
- `backend/`: Golang 后端项目

## 快速开始

### 后端 (Backend)

1. 进入 `backend` 目录:
   ```bash
   cd backend
   ```
2. 运行服务器:
   ```bash
   go run main.go
   ```
   服务器将在 `http://localhost:8080` 启动。

### 前端 (Frontend)

1. 进入 `frontend` 目录:
   ```bash
   cd frontend
   ```
2. 安装依赖:
   ```bash
   npm install
   ```
3. 启动开发服务器:
   ```bash
   npm run dev
   ```
